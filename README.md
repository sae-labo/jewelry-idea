# Jewelry Idea

自分のジュエリーコレクションと、これから欲しいアイテムを整理するための個人プロジェクトです。

## このプロジェクトで管理すること

- 欲しいジュエリー
- 所有しているジュエリー
- 購入の優先順位
- リングの着用位置
- コーディネートや重ね付けのアイデア
- 婚約指輪・結婚指輪の候補
- 将来の購入計画

## ファイル

- [wishlist.md](wishlist.md)  
  欲しいジュエリーと優先順位

- `owned-jewelry.md`  
  所有しているジュエリー

- `purchase-plan.md`  
  予算や購入時期の計画

- `ideas/`  
  リング、ネックレス、ピアスのアイデア
